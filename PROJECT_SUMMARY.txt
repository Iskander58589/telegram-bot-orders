🚀 БЫСТРЫЙ СТАРТ (5 МИНУТ)
════════════════════════════════════════════════════════════════

1️⃣  УСТАНОВКА ЗАВИСИМОСТЕЙ (1 минута)
   $ pip install -r requirements.txt

2️⃣  ПОЛУЧЕНИЕ ADMIN_CHAT_ID (3 минуты) - ⚠️ ВАЖНО!
   $ python debug_tools.py
   Выберите опцию 1 и получите ID админ-группы

3️⃣  КОНФИГУРАЦИЯ (1 минута)
   • Откройте config.py
   • Замените ADMIN_CHAT_ID = -1001234567890 на ваш ID
   • Сохраните файл

4️⃣  ЗАПУСК БОТА (1 минута)
   $ python bot.py
   
   Вы должны увидеть: "🚀 Бот запущен!"

5️⃣  ПРОВЕРКА (1 минута)
   • Откройте Telegram
   • Найдите вашего бота
   • Напишите /start
   • Должны появиться кнопки!

  PAYMENT_DETAILS = """💳 Ваши реквизиты"""

ОТРЕДАКТИРУЙТЕ СООБЩЕНИЯ:

После изменений просто перезагрузите бота!

🐛 ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ
════════════════════════════════════════════════════════════════

ОШИБКА: "ModuleNotFoundError: aiogram"
→ РЕШЕНИЕ: pip install -r requirements.txt

ОШИБКА: "Заказы не приходят в админ-группу"
→ РЕШЕНИЕ:
  1. Проверьте ADMIN_CHAT_ID в config.py
  2. Убедитесь, что бот администратор в группе
  3. Запустите: python debug_tools.py
  4. Прочитайте README.md раздел Troubleshooting

ОШИБКА: "Токен неверный"
→ РЕШЕНИЕ: Проверьте BOT_TOKEN в config.py
  (он уже установлен корректно)

БОТ НЕ ОТВЕЧАЕТ:
→ РЕШЕНИЕ:
  1. Перезагрузите бота (Ctrl+C + python bot.py)
  2. Проверьте интернет
  3. Используйте python debug_tools.py для диагностики

🔄 ЖИЗНЕННЫЙ ЦИКЛ ЗАКАЗА
════════════════════════════════════════════════════════════════

ПОЛЬЗОВАТЕЛЬ:
  1. Нажимает /start → видит главное меню
  2. Выбирает "Заказать"
  3. Выбирает тип работы
  4. Выбирает срочность
  5. Пишет детали заказа
  6. Видит итоговую цену
  7. Подтверждает заказ
  8. Видит реквизиты оплаты
  9. После оплаты нажимает "Я оплатил"

АДМИНИСТРАТОР:
  1. Получает уведомление о новом заказе в группе
  2. Выполняет работу вручную
  3. Отправляет результат клиенту
